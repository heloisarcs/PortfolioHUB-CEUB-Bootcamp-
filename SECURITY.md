# Política de Segurança do PortfolioHUB

## Relatório de Vulnerabilidades
Agradecemos a colaboração em reportar vulnerabilidades. Por favor, use a seção **Issues** do GitHub para relatar bugs ou falhas de segurança.

## Ramificação (Branch Protection)
A branch principal (`main`) é protegida. [cite_start]Qualquer alteração deve ser enviada via **Pull Request (PR)** e revisada por um colaborador antes do merge, garantindo o controle de versão.

# Guia de Colaboração (CONTRIBUTING)

Quer contribuir para o PortfolioHUB?

1. **Fork o Repositório.**
2. **Crie uma nova Branch:** Use nomes descritivos (ex: `feature/nova-secao`).
3. **Faça Commits:** Use mensagens claras (ex: `fix: corrige erro no estilo`).
4. **Abra um Pull Request (PR):** Envie o PR para a branch `main` e aguarde a revisão.
